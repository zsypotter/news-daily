# 每周GitHub Star飙升AI项目

这个文件夹用于记录每周GitHub上Star数目飙升的AI相关项目。

## 目录结构

按周创建文件，格式为 `YYYY-WW.md`（例如：2026-08.md 表示2026年第8周）

## 内容要求

每个项目应包含以下信息：
- **项目名称**：GitHub仓库名称
- **项目链接**：GitHub仓库URL
- **Star数目**：当前Star数及本周增长量
- **项目简介**：一句话描述项目功能和核心亮点
- **技术标签**：如 [LLM], [计算机视觉], [语音识别], [开源工具] 等

## 示例

```markdown
# 2026-08 每周Star飙升AI项目

## 1. Awesome-LLM
- 🔗 GitHub: https://github.com/...
- ⭐ Star: 12000+（本周增长3000+）
- 📝 简介：收集了最新的LLM模型、论文、工具和资源的综合列表
- 🏷️ 标签: [LLM], [资源汇总], [开源]

## 2. Vision-Language-Model
- 🔗 GitHub: https://github.com/...
- ⭐ Star: 5000+（本周增长2500+）
- 📝 简介：结合视觉和语言的多模态AI模型，支持图像描述、视觉问答等任务
- 🏷️ 标签: [多模态], [计算机视觉], [自然语言处理]
```
